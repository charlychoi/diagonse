# 마케팅 사전진단 보고서 목차 (자동생성용)

```markdown
# 마케팅 사전진단 보고서 — {회사명}

> 생성형 AI 스킬(diagonse) · URL · 회사명 · 생성 시각

## 1. Cover
- URL, 회사명, 키워드, 점수 요약

## 2. Executive Summary
- 표면 점수 / 브랜드–서비스 연결 / 기술 가이드 점수 구분
- 한 줄 문제 + 처방

## 3. 브랜드 검색 목표 · 문제 · 부작용
- 목표 쿼리 (브랜드 단독 / 브랜드+서비스)
- 현재 연결 강도
- root causes
- 전략 단계

## 4. 표면 축 Score Card
- brand / contentSeo / uxConversion / socialPaid / authorityAi
- 각 축: 무엇을 봤는지 / 실측이 아닌 것

## 5. 실측 HTML (Before)
- title, description, H1, OG, canonical, robots 요약

## 6. 신호 매트릭스
- 브랜드·서비스·전문 × title/desc/H1

## 7. Before → After 문안
- title / meta / H1 / OG / 히어로
- 각 변경의 브랜드 검색 효과

## 8. 네이버 서치어드바이저 기술 전제
- robots(Yeti), noindex, canonical, HTTPS, OG, 구조화 데이터 등
- 공식 가이드 링크

## 9. 실행 체크리스트 · 대행사 지시
- 실검색 캡처 → 문안 교체 → 채널 정렬 → 재검색

## 10. Methodology · 한계
- 순위 미보장
- 표면 ≠ 구조(6대) 실측
```
