# AI 온라인 마케팅 사전진단 사용자 설명서

## 접속

공개 사이트: [https://diagnose.charlychoi.chatgpt.site](https://diagnose.charlychoi.chatgpt.site)

## GitHub 복제 후 로컬 실행

프로젝트 폴더에서 다음 명령을 실행합니다.

```bash
npm install
cp .env.example .env.local
npm run dev
```

`.env.local`에 본인의 `OPENAI_API_KEY`를 입력한 뒤 `http://127.0.0.1:3000`을 엽니다. 키 파일은 Git에 커밋하지 않습니다.

## 진단 순서

1. 회사 홈페이지 URL과 회사명을 입력합니다.
2. 필요하면 키워드·업종·경쟁사 URL을 추가합니다.
3. **AI 사전진단 시작하기**를 누릅니다.
4. AI 전략과 경쟁사 비교를 확인합니다.
5. Markdown·HTML·PDF로 저장합니다.

AI 웹 검색이 포함되어 수십 초 정도 걸릴 수 있습니다.

## 경쟁사 분석

경쟁사 URL을 비워두면 GPT-5.6이 실시간 웹 검색으로 같은 업종·서비스·지역의 공식 HTTPS 홈페이지를 최대 3개 찾습니다. 이후 각 홈페이지를 실제 수집해 공개 표면 신호를 비교합니다.

직접 경쟁사 URL을 입력하면 사용자 입력을 우선합니다. 비교는 매출·광고 성과·평판이 아니라 공개 홈페이지의 CTA, 문의 경로, 콘텐츠, 구조화 데이터 등을 기준으로 합니다.

## 문제 해결

- `OPENAI_API_KEY가 설정되지 않았습니다`: 프로젝트의 `.env.local`을 확인하고 앱을 다시 실행합니다.
- `OpenAI API 오류`: API 키 상태, 사용 한도, 네트워크를 확인합니다.
- 일부 경쟁사 수집 실패: 해당 경쟁사만 제외하고 나머지 비교는 계속됩니다.
- AI 오류가 나도 기본 진단 완료: 정상적인 안전 폴백이며 오류 사유가 결과에 표시됩니다.

## 종료

실행 중인 터미널에서 `Control + C`를 누릅니다.
